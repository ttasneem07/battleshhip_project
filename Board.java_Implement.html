Board.java Implementation

Board UML

(Do not begin this class until you have completed and tested the Ship and Square classes!)

The Board class is a model template for the human player’s and computer’s boards in Battleship. It uses a 2D array of Square objects to represent the squares in the board. Note: you may not alter the method signatures in any way, or the Battship game will not work properly.

The following are the instance variables (fields) and methods of the Board class. Complete each item in the following order.

Instance Variable (Fields)

Declare the following private instance variables (fields) (see UML for naming):

  • An integer that represents the number of rows in the 2-dimensional array of Squares
  • An integer that represents the number of columns in the 2-dimensional array of Squares
  • A 2-dimensional array of Squares
  • An integer that represents that maximum number of Ships on the board
  • An array of Ships
  • An integer that represents the number of Ships on the board

Methods

The following is the list of constructors and methods you must complete.

  • public Board(int numberOfRows, int numberOfColumns, int maximumNumberOfShips): This is the constructor of the class.
    • This constructor must initialize the class’s instance variables using the parameter values as appropriate.
    • Throw an IllegalArgumentException with the message, “Invalid rows”, if numberOfRows is less than 1.
    • Throw an IllegalArgumentException with the message, “Invalid cols”, if numberOfColumns is less than 1.
    • Throw an IllegalArgumentException with the message, “Invalid ships”, if maximumNumberOfShips is less than 1.
    • NOTE: You must check for these errors in the order given above.
  • public int getNumberOfRows(): Returns the number of rows in the Board.
  • public int getNumberOfColumns(): Returns the number of columns in the Board.
  • public int getNumberOfShips(): Returns the number of Ships on the Board.
  • public boolean addShip(int length, boolean isHorizontal, int startRow, int startCol): Creates a new Ship with the given length and adds it to the array of Ships as well as the Board. This means that all Squares the length of the Ship starting at the startRow and startCol must be updated appropriately. The isHorizontal parameter tells which direction to fill Squares. The method returns true if the Ship is successfully added to the Board and false if the Ship could not be added due to another Ship in the way, reaching a Board boundary, or if the array of Ships is full. If the Ship cannot be placed, it should not be added to the array of Ships and none of the Squares should be changed.
    • Throw an IllegalArgumentException with the message, “Invalid length”, if length is less than 1.
    • Throw an IllegalArgumentException with the message, “Invalid row”, if startRow is less than 0 or startRow is greater than or equal to the number of rows.
    • Throw an IllegalArgumentException with the message, “Invalid col”, if startCol is less than 0 or startCol is greater than or equal to the number of columns.
    • NOTE: You must check for these errors in the order given above.
  • public Ship getShip(int row, int col): Returns the Ship contained in the given row and column of the Board; this may be null if there is no Ship at that location.
    • Throw an IllegalArgumentException with the message, “Invalid row”, if row is less than 0 or row is greater than or equal to the number of rows.
    • Throw an IllegalArgumentException with the message, “Invalid col”, if col is less than 0 or col is greater than or equal to the number of columns.
    • NOTE: You must check for these errors in the order given above.
  • public Ship[] getShips(): Returns the array of Ships.
  • public boolean fireAtLocation(int row, int col): If the Square located at the specified row and column has not been previously hit, the Square should be fired at and true should be returned. If the Square has already been hit, it should not be hit again and false should be returned.
    • Throw an IllegalArgumentException with the message, “Invalid row”, if row is less than 0 or row is greater than or equal to the number of rows.
    • Throw an IllegalArgumentException with the message, “Invalid col”, if col is less than 0 or col is greater than or equal to the number of columns.
    • NOTE: You must check for these errors in the order given above.
  • public boolean hasBeenHit(int row, int col): Returns true if the Square located at the specified row and column has been fired upon, false otherwise.
    • Throw an IllegalArgumentException with the message, “Invalid row”, if row is less than 0 or row is greater than or equal to the number of rows.
    • Throw an IllegalArgumentException with the message, “Invalid col”, if col is less than 0 or col is greater than or equal to the number of columns.
    • NOTE: You must check for these errors in the order given above.
  • public boolean areAllShipsSunk(): Returns true if all of the Ships on the Board have been sunk, false otherwise.
  • public String toString(): Returns a String representation of a Board as shown below.
    Use the toString() method of the Square class to get the String representation for each Square of the board. There must be space between the Strings for each Square, but no space at the end of each row/line.
